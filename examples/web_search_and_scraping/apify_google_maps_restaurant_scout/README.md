# Restaurant Scout Agent

An AI agent that searches **Google Maps** for restaurants and food spots using natural language queries. Built with the **Upsonic AI Agent Framework** and **Apify**. Ask it anything like *"cheap falafel in Kadıköy"* or *"vegan brunch in Cihangir"* and get a curated list saved to Markdown.

## What It Does

1. **Receives** a natural language query describing what and where to find
2. **Searches** Google Maps via Apify's `compass/crawler-google-places` Actor
3. **Interprets** the results and formats a clean, readable response
4. **Saves** the output to `results.md`

## Quick Start

### 1. Install Dependencies

```bash
cd examples/web_search_and_scraping/apify_google_maps_restaurant_scout

# With uv (recommended)
uv venv && source .venv/bin/activate
uv pip install -r requirements.txt

# With pip
python3 -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
```

### 2. Set Up API Keys

```bash
cp .env.example .env
```

Edit `.env` with your keys:

```bash
APIFY_API_KEY=your_apify_api_token_here      # Free tier at console.apify.com
ANTHROPIC_API_KEY=your_anthropic_api_key_here
```

### 3. Run

```bash
python main.py
```

Results are printed to the terminal and saved to `results.md`.

## How It Works

### Apify Tools Used

| Actor | Purpose |
|-------|---------|
| `compass/crawler-google-places` | Searches Google Maps and returns place details including name, address, rating, price level, and reviews |

`ApifyTools` registers the Actor as a callable tool and automatically exposes its full input schema to the agent, so it knows exactly which parameters to pass based on the user's query. See the [Upsonic Apify integration docs](https://docs.upsonic.ai/concepts/tools/scraping-tools/apify#apifytools) for full configuration options.


## Customization Guide

### Change the Query

Edit `main.py`:

```python
task = Task("Best rooftop bars in Beyoğlu, Istanbul")
```

### Swap the Actor

The Google Maps Actor is just one of thousands available on the [Apify Store](https://apify.com/store). You can swap it for any other Actor by changing the `actors` field:

```python
ApifyTools(
    actors=["apify/instagram-scraper"],  # browse more at apify.com/store
    actor_defaults={
        "apify/instagram-scraper": {
            "directUrls": ["https://www.instagram.com/some_account/"],
        }
    },
    timeout=180.0,
    max_retries=0,
)
```

Each Actor has its own input schema. Upsonic's `ApifyTools` fetches it automatically, so the agent always knows which parameters to pass. You can also interact with Actors directly via the [Apify API](https://docs.apify.com/platform/integrations/api) if you need lower-level control.

### Use a Different Model

```python
agent = Agent(
    "anthropic/claude-sonnet-4-5",  # or "openai/gpt-4o"
    tools=[...]
)
```

### Adjust Result Count

```python
"maxCrawledPlacesPerSearch": 5,   # Fewer, faster
"maxCrawledPlacesPerSearch": 20,  # More results (watch token limits)
```

## Notes

- Each run typically takes 60-90 seconds as the Google Maps Actor needs time to crawl
- Apify's free tier is enough to run several searches
- Keep `maxCrawledPlacesPerSearch` at 10 or below since more results can exceed the model's context limit
- Output is saved as `results.md` in the current directory
